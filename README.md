# traceback-platform
A community-driven open-source proposal designed to streamline device theft reporting and provide actionable intelligence to An Garda Síochána.

Frontend scaffold

This repo includes a minimal Vite + React frontend scaffold for the registration form demo.

Quick start (from repository root):

```bash
cd traceback-platform
npm install
npm run dev
# open http://localhost:5173
```

Files of interest:
- `src/features/auth/RegistrationForm.jsx` — the registration component
- `src/styles/tokens.css` — CSS variables used by the form
- `index.html`, `tailwind.config.js`, `postcss.config.js` — Vite/Tailwind setup
